===============
savi2mkv
===============

What it does.
==============

Simple script to transform an AVI video to MKV.


What I need.
=============

To use PhotoShare, you need at least:

* Python (>=2.7)
* mkvtools

How to use
===========

To run, you only need to execute photoshare.py

``$ ./savi2mkv --help``
