===============
avi2mkv
===============

What it does.
==============

Simple script to transform an AVI video to MKV.


What I need.
=============

To use avi2mkv, you need at least:

* Python (>=2.4)
* mkvtools

How to use
===========

To run, you only need to execute photoshare.py

``$ ./avi2mkv --help``
